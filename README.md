# Data repository

Repository containing the mesh and image files of the numerical examples solved in the article _A new robust formulation for optical-flow/material identification problems_, by authors: Jorge M. PÃ©rez Zerpa, Gonzalo D. Maso Talou and Pablo Blanco.

For each example and test case the following files are included:

- deformed meshes (vtk format)

- images (vtk format)


The pattern used for the filenames of intensity is:

[problemName]\_intensities\_nIm\_[valuenIm]\_Dp\_[valueDp]
\_errorlevel\_
